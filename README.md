# 🧮 Calculator

A simple and responsive Calculator built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean and user-friendly interface.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear (AC) Button
- 🖥️ Responsive UI
- ⚡ Fast and lightweight
- 🎨 Modern design

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

---

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/alokyadav-dev/calculator.git
```

2. Open `index.html` in your browser.
